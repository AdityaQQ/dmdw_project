# 📊 DMDW Machine Learning Project

A comprehensive Machine Learning project demonstrating multiple algorithms including KNN, K-Means, K-Medoids, and DBSCAN, with visualization and performance comparison.

---

## 🚀 Overview

This project applies various machine learning techniques on a dataset to:

* Perform classification and clustering
* Compare algorithm performance
* Visualize results using plots and confusion matrices
* Understand real-world data behavior

---

## 🧠 Algorithms Implemented

* 🔹 K-Nearest Neighbors (KNN)
* 🔹 K-Means Clustering
* 🔹 K-Medoids
* 🔹 DBSCAN
* 🔹 Linear Regression

---

## 📊 Key Features

* 📈 Data preprocessing & normalization
* 📉 Visualization of clusters
* 🔍 Confusion Matrix analysis
* 📊 Performance comparison between models
* 🖼️ Graphical outputs saved automatically

---

## 🛠️ Tech Stack
